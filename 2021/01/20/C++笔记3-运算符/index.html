C++笔记-第三章-运算符

发表于 2021-01-20 | 分类于 编程

运算符的作用：用于执行代码的运算

运算符的类型：算术运算符、赋值运算符、比较运算符、逻辑运算符

一、算术运算符

作用：处理四则运算

算术运算符包括以下的几种符号：
1.加号+
2.减号-
3.乘号*
4.除号/
5.取模号%
6.自增++
7.自减--

注意：取模号%只能对整数取模；自增自减有两种，如果变量在前，则先使用变量的值，再加一，否则先加一，后使用变量的值

二、赋值运算符

作用：将表达式的值赋给变量

赋值表达式有两种，一种是等于号=；另一种是一种算术运算符加上等于号。

  • 1.等于号 = 没啥好说的，就像是a = 6
  • 2.例如a *= 3，他等同于a = a * 3，其余的类型同理

三、比较运算符

作用：用于表达式的比较，返回一个真或假的值(0或1)

比较运算符有如下几种：

  • 相等于 ==
  • 不等于 ！=
  • 大于 >
  • 小于 <
  • 大于等于 >=
  • 小于等于 <=

四、逻辑运算符

逻辑运算符有三种：

  • 非运算符！(如果a为假，则！a为真；如果a为真，则！a为假)
  • 与运算符&&(只有a和b都是真，结果才为真)
  • 或运算符||(a或b有一个真，结果就是真的)

本章结束，感谢观看！
