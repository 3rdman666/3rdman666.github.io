C++笔记-第七章-指针

发表于 2021-01-23 | 分类于 编程

一、指针的基本概念

指针是用来间接访问内存的新变量

  • 内存编号从0开始，一般用十六进制表示
  • 可以用指针变量保存地址

二、指针变量的定义和使用

指针变量定义语法：数据类型 *变量名
当然，也可以在定义的时候直接指向变量地址，也就是：数据类型 *变量名 = &另一变量名

这里的’&’是地址符号，指的是后面的变量的地址

三、空指针和野指针

空指针：指向内存编号为0的空间的指针变量，用来初始化指针变量，但是空指针指向的内存不可以访问
语法：数据类型 *变量名 = NULL
野指针：指向非法内存空间的指针变量

四、const修饰指针

有三种情况：

  • 1.const修饰指针–>常量指针
    语法：const int *p = &a
    指针的指向可以修改，但是指针指向的值不可以修改
  • 2.const修饰常量–>指针常量
    语法：int * const p = &a
    指针的指向不可以改，但是指针指向的值可以修改
  • 3.const即修饰指针又修饰常量
    语法：const int * const p = &a
    指针的指向和指向的值都不可以修改

TIPS(指针的一些技巧):
1.指针可以访问数组中而的元素，数组首地址就是数组名，可以使用指针指向该地址
2.指针变量可以作为函数参数使用
3.可以用一个指针变量指向另一个指针变量(神tmd套娃)
