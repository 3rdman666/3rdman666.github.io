个人博客的搭建方法

发表于 2021-01-19 | 分类于 博客搭建

步骤一：一些准备工作

我们需要两个插件：一个是Git，另一个是Nodejs，下载链接如下：

Git点击此处下载
Nodejs点击此处下载

下载完成之后安装，安装完成之后，我们要验证我们的Nodejs软件是否安装成功：
1.进入命令提示窗，或者点击桌面右键选择Git Bash Here。

2.输入以下命令来验证您的Nodejs是否安装成功：

npm -v

如果屏幕提示版本号，则说明Nodejs安装成功！

3.由于自家网速较慢，我们需要换源，这里换的是国内阿里巴巴的源，输入下面的命令来完成换源：

npm config set registry https://registry.npm.taobao.org

4.在完成安装Nodejs的前提下，输入以下命令来安装最重要的Hexo！！！

npm install hexo-cli -g

然后输入命令：

hexo -v

如果屏幕提示版本号，则说明Nodejs安装成功！

5.至此，准备工作完成，开始进行下一步的工作。

步骤二：开始生成本地博客文件

在这之前，再你的电脑磁盘中找到一个地方，并创建一个新的文件夹，这个文件夹是用来存放你博客的本地文件的。文件夹命名最好是英文，不要有中文。

然后进入该文件夹，这里面是空的，什么都没有。然后对着空白处右键选择Git Bash Here，然后输入下面的命令来生成博客文件：

hexo init

稍作等候，文件夹内充满了博客文件。

接下来开始在本机开启博客网站：
1.输入如下命令开始编译本地博客文件：

hexo g

2.输入如下命令开始运行博客网站：

hexo s

3.打开浏览器，访问localhost:4000即可看到我们的博客网站了。
TIPS：如果想关闭博客，只需按下Ctrl＋C就行了。

这种开启博客的方法只能由自己看到，而别人看不到。那么怎么样让别人看到自己的博客呢？

步骤三：将本地博客文件部署到服务器（1.创建Github仓库）

部署到服务器，可以让别人访问我们的博客，在这之前，我们需要用到一个重要的东西：Github！

你需要一个Github账号，去官网注册一个：Github官网

然后创建一个仓库：点击右上角个人图标，选择Your Repositories，他会让你填写仓库名称，这里的仓库名称必须按照如下格式填写！！！！！！！

你的Github账户名称.github.io

例如我的账户名称叫做zhangsan

那么我的仓库名应该是：

zhangsan.github.io

步骤四：将本地博客文件部署到服务器（2.将本地文件上传到仓库）

回过头来，我们来到自己博客的根目录文件夹，右键选择Git Bash Here，输入下面的命令：

npm install hexo-deployer-git --save

这是安装hexo-deployer-git插件的指令，是将本地文件上传到服务器的必要插件。

然后打开你的GitHub仓库，你会看到有两个协议，一个是HTTPS，另一个是SSH，这里我选择HTTPS作为例子讲解。

填写全之后，回到gitbash画面，输入下面两行命令：

git config --global user.name '你的GitHub用户名'

git config --global user.email '你的邮箱地址'

紧接着输入下面的命令：

hexo g -d

稍等片刻，会弹出来一个GitHub的窗口，它会要求你输入一个令牌。

然后我们来到GitHub网站，点击右上角头像，点击Settings，再点击下面的Developer settings，进去之后点击Personal access tokens来创建自己的令牌，创建令牌他会给你令牌的内容，复制粘贴到前面的GitHub窗口上。

然后稍等片刻，它会提示deploy done git,说明上传成功。

在浏览器上输入你的仓库名就可以访问你的博客了！
