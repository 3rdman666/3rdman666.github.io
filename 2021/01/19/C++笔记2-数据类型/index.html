三、实型

作用；用来表示小数
浮点类型变量分为两种：

  • 1.单精度float
  • 2.双精度double

区别在于表示的有效数字范围不同，float占用4字节空间，有效数字范围是7位有效数字；而double占用8字节的内存空间，有效数字范围是15~16位有效数字

四、字符型

作用：字符型变量用于显示单个字符

语法：char ch = 'a'

  • 注意1：用单引号括起来，不要用双引号
  • 注意2：单引号只能有一个字符，不能是字符串

C++字符型变量占用1字节，而且字符型变量不是把字符本身存入内存，而是存入相应的ASCLL码

六、字符串类型

作用:用来表示一串字符
有两种风格的字符串表达方式：

  • 1.C风格字符串-定义方式：char 变量名[] = "字符串"
  • 2.C++风格字符串-定义方式：string 变量名 = "字符串"

TIPS：使用C++风格字符串时，需要添加头文件 #include < string >

七、布尔类型(bool)

作用：代表真或假的值

bool类型只有两个值：

  • true—真(本质是1)
  • false—假(本质是0)
